## Aktuelle Features (Stand: Januar 2026)

- **Globaler Header**: Auf allen Screens sichtbar (außer Loading)
- **Einheitliche Abstände**: 28px links/rechts für konsistente Ausrichtung
- **"Übersicht"**: Dashboard mit Quick Actions, Tasks, Ziele
- **Swipe-Navigation**: Finger-Gesten für Vor/Zurück
- **SVG-Icons**: Konsistente Stroke-based Icons
- **Gradient-Buttons**: 30px border-radius
