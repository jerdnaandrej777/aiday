# Contributing

Voraussetzungen
- Node/Deno, Supabase CLI, Postman (optional).

Setup
- Kopiere .env.example -> .env, fülle Variablen.
- supabase start && supabase db reset.

Entwicklung
- Edge Functions mit supabase functions serve <name>.
- Deno fmt/lint/typecheck vor Commit.

Richtlinien
- Kleine PRs, aussagekräftige Commits, Code-Reviews.
